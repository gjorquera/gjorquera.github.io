List Types

Ordered list:

  1. First item
  2. Second item
  3. Third item

Unordered list:

  • List item
  • Another item
  • And another item

Nested list:

  • Item
    1. First Sub-item
    2. Second Sub-item

Math

Inline math: \(\phi = \frac{1 + \sqrt{5}}{2} = 1.6180339887\dotso\)

Block math:

$$\phi = 1 + \frac{1}{1 + \frac{1}{1 + \frac{1}{1 + \dotso}}}$$
